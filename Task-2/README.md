# Fiaba Bakery Website

A responsive bakery website created using HTML and CSS with a red and cream color scheme.

## Project Overview

Fiaba Bakery is a fictional bakery website showcasing bakery products, information about the owners, and contact details. The website features a warm and inviting design with a carefully selected red and cream color palette that reflects the bakery's brand identity.

## Features

- **Responsive Design**: Adapts to mobile, tablet, and desktop screens
- **Modern Layout**: Clean, visually appealing interface
- **Color Scheme**: Consistent use of red (#C41E3A) and cream (#FFFDD0) throughout
- **Multiple Sections**:
  - Navigation bar with logo and menu
  - Hero banner with bakery introduction
  - About section featuring the two owners
  - Product showcase gallery
  - Contact information and location
  - Footer with social media links

## Color Scheme

The website uses the following color palette:

- **Primary Red**: `#C41E3A` (Used for headers, buttons, and accents)
- **Cream Background**: `#FFFDD0` (Main background color)
- **Dark Text**: `#333333` (Primary text color)
- **Light Text**: `#666666` (Secondary text color)
- **White**: `#FFFFFF` (Text on red elements)

## Typography

- **Headings**: "Playfair Display", serif - Elegant and sophisticated
- **Body Text**: "Open Sans", sans-serif - Readable and clean

## Setup Instructions
